# Daily Planner 开发说明

## 进入项目根目录

**PowerShell（Cursor 默认终端）：**
```powershell
cd d:\cursor1\workspace\daily-planner
```

**CMD 命令提示符：**
```cmd
cd /d d:\cursor1\workspace\daily-planner
```

## 启动项目

```bash
npm install
npm run dev
```

浏览器打开：http://localhost:5173

## 情绪树洞

- 无后端，前端直连千问 API
- 在「设置」里填写千问 API Key 后即可使用
